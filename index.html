HTML and CSScoding a website

Greg Schneider

My first semester at BYU-Idaho I was a physics major with a geology emphasis. Then I decided to switch the two and became a Geology student with a physics minor. I took an introductory GIS course and loved it, ultimately switching degrees again to focus on it.

Why Code a Website?

The Geospatial Computing degree at BYU-Idaho really feels like an Interdisciplinary Degree,but more focused and with a better name. As a student I have taken courses in geology, math, statistics, GIS, and programming. I've completed some very in-depth and rigorous projects in every subject except programming.

My goal with this project was to do apply the coding skills I'd learned from my courses thus far and to teach myself CSS, which I hadn't covered in any previous coursework. At my current job I have come in contact with it several times already and have not been able to perform the task at hand as well as I would like.

The true outcome of this website is not really the content seen on the webpages (although I did try and make it meaningful and applicable to anyone who may come across it), but the HTML and CSS code working behind the scenes. The code and the powerpoint presentation discussing it will be available on the Documents page for download.
